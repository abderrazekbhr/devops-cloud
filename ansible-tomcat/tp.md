# TP Tomcat Avec Ansible

1) Ecrire un playbook qui permet d'installer et de configurer Tomcat(utiliser l'inventaire Yaml et utiliser les variables).

2) Ecrire un playbook Ansible qui permet de déployer une application sur tous les serveurs d'application Tomcat sur un Virtual host particulier avec la méthode du fichier xml.

3) Créer un role Tomcat en se basant sur le playbook d'installation et de configuration en utilisant la structuration de role d'ansible-galaxy.
